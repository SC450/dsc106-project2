Proposition: NYPD officers are more often cleared of allegations than found guilty, with incrimination rates remaining proportional over time.

FOR the Proposition

Design Decisions and Rationale:

  • Title and description: I made sure to use language that would be in support of the proposition. Using the phrase "no one is above the law" creates an engaging and eye-catching title, and I honestly gave a description of the plot in the description. I would rate this decision's earnestness as 1.5.
  • Annotations: I really wanted this visualization to agree with the proposition's point that both lines grow proportional to each other, so I emphasized the proportional growth with orange arrows, which show the scalar by which you can multiply the value at the red line to reach the value at the blue line. Additionally, I added text which restates that both complaint types accumulate proportionally to each other. I would rate this decision's earnestness as 2.
  • Color: I chose colors that would stand out from each other and be unique. The blue color for the "Not Substantiated" line is a calm and neutral color, while the red color for the "Substantiated" line is a more aggressive color. I chose the arrows to be orange, as orange is distinguishable from both blue and red, and clearly emphasizes the proportionality between the two lines. I would rate this decision's earnestness as 2.

AGAINST the Proposition

Design Decisions and Rationale:

  • Grouping and aggregation: I created the "No Action Taken" and "Arrest/Summons Issued" categories by grouping similar values together for each category. For example, values in the "Arrest/Summons Issued" category include arrests of any kind, as well as summons of any kind. This makes the category of NYPD officers who have done wrongdoing look larger than it would be if I had separated the arrests and summons into two separate categories. I would rate this decision's earnestness as -1.
  • Title and description: I made sure to use negative language in my title to persuade the reader that more recent years have seen a greater number of NYPD officers being punished for their actions. Using the phrase "modern generations of NYPD officers are failing themselves" catches the reader's attention by utilizing a negative connotation. I also stated in my description that the rate of complaints in which there is no action taken has "fallen behind", emphasizing the idea that NYPD officers have become more troublesome over time. I would rate this decision's earnestness as -1.
  • Annotations: I wanted to emphasize that modern generations of officers have been more problematic than officers of past years, so I annotated the plot with an orange line at the year 2005, which is when the "Arrest/Summons Issued" line overtakes the "No Action Taken" line. I also added an arrow and text annotation to make this point clear. The orange line serves a "divisor" between the older and newer years, and makes it easy to see that both lines have clearly not been growing proportionally to each other. I would rate this decision's earnestness as 1.5.
